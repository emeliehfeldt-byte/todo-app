# To Do-App
A simple task management application built with HTML, CSS and JavaScript.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Local storage support

## Technologies Used
- HTML
- CSS
- JavaScript (ES6)
- Browser Local Storage API

## What I Learned

DOM manipulation

Event handling in JavaScript

State management using local storage

Structuring a small front-end project

Basic UI/UX design principles

## Future Improvements

Due dates

Filtering (completed / active)

Backend integration (Node.js / API)
